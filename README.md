## Instalação

1. Clonar o repositorio
   ```sh
   git clone https://github.com/JecianeSilva/Pokedex.git
   ```
2. Entrar na pasta do projeto
   ```sh
   cd pokedex
   ```
3. Instalar as dependências
   ```sh
   yarn or yarn install
   ```
4. Rodar o projeto com Expo
   ```js
   expo start or yarn start
   ```
